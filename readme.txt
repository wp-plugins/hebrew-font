Hebrew Font plugin provides an easy way to use Hebrew fonts on your WordPress website.

== Description ==

Hebrew Font plugin allows you to select a Hebrew font from a variety of additional Hebrew fonts.

Using Hebrew fonts in your website is now easy! <br />
Just choose the font you like from the Font-Family dropdown which exists in the visual editor.

= Note: =
The Font-Family dropdown contains fonts for both: English and Hebrew. There are fonts which change only English letters.

<br />

== Installation ==

1. Download this plugin directly from your Wordpress Admin Page.
1. Click Install.
1. Click Activate.
